# Skeleton

Loading placeholder with shimmer animation

## Usage

### Web Component

```html
<ds-skeleton>Content</ds-skeleton>
```

### React

```tsx
import { Skeleton } from "@ds/react";

function Example() {
  return <Skeleton>Content</Skeleton>;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content to display |

## Accessibility

- **Keyboard**: N/A (non-interactive)
- **ARIA**: role=status, aria-busy, aria-label
- **Screen reader**: Uses role=status with aria-busy=true to announce loading state. Provides accessible label (default 'Loading...'). Screen readers announce the loading status without repeated announcements.
