# Tag

Removable tag/chip for categories and selections

## Usage

### Web Component

```html
<ds-tag>Content</ds-tag>
```

### React

```tsx
import { Tag } from "@ds/react";

function Example() {
  return <Tag>Content</Tag>;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content to display |

## Accessibility

- **Keyboard**: Enter, Space, Backspace, Delete
- **ARIA**: role=status, role=button, aria-disabled, aria-label
- **Screen reader**: Non-clickable tags use role=status. Clickable tags use role=button with keyboard activation. Remove button has accessible 'Remove' label. Disabled state announced via aria-disabled.
