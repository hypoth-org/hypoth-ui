# Separator

Visual divider between content sections

## Usage

### Web Component

```html
<ds-separator>Content</ds-separator>
```

### React

```tsx
import { Separator } from "@ds/react";

function Example() {
  return <Separator>Content</Separator>;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content to display |

## Accessibility

- **Keyboard**: N/A (non-interactive)
- **ARIA**: role=separator, role=none, aria-orientation
- **Screen reader**: Uses role=separator with aria-orientation for semantic separators. Decorative separators use role=none to be ignored by assistive technology. Non-decorative separators create logical content boundaries.
