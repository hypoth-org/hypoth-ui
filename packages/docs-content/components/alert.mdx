# Alert

Contextual feedback message with status variants

## Usage

### Web Component

```html
<ds-alert>Content</ds-alert>
```

### React

```tsx
import { Alert } from "@ds/react";

function Example() {
  return <Alert>Content</Alert>;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Content to display |

## Accessibility

- **Keyboard**: Escape
- **ARIA**: role=alert, role=status, aria-live=assertive, aria-live=polite
- **Screen reader**: Error and warning variants use role=alert with aria-live=assertive for immediate announcement. Info and success variants use role=status with aria-live=polite. Closable alerts have an accessible dismiss button.
