# Acme Integration Guide

This guide covers Acme-specific integration patterns and best practices.

## Getting Started

To use the Acme Design System in your application:

1. Install the package:
   ```bash
   npm install @acme/design-system
   ```

2. Import the Acme theme:
   ```javascript
   import '@acme/design-system/themes/acme.css';
   ```

3. Start using components:
   ```html
   <ds-button variant="primary">Get Started</ds-button>
   ```

## Brand Tokens

Acme uses a custom set of design tokens:

| Token | Value | Usage |
|-------|-------|-------|
| `--acme-primary` | `#7c3aed` | Primary actions, links |
| `--acme-secondary` | `#64748b` | Secondary UI elements |
| `--acme-accent` | `#f59e0b` | Highlights, warnings |
