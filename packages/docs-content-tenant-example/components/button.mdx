# Button

The Acme Design System button follows our brand guidelines with custom styling.

## Acme Usage

Our button component is pre-configured with Acme brand colors:

```html
<ds-button variant="primary">Acme Primary</ds-button>
<ds-button variant="secondary">Acme Secondary</ds-button>
```

## Brand Guidelines

When using buttons in Acme applications:

- **Primary buttons** use our signature purple (#7c3aed)
- **Secondary buttons** use subtle gray tones
- Avoid using more than one primary button per section

## Custom Properties

Acme buttons expose these CSS custom properties for fine-tuning:

| Property | Description | Default |
|----------|-------------|---------|
| `--acme-button-radius` | Border radius | `8px` |
| `--acme-button-shadow` | Box shadow | `none` |
| `--acme-button-transition` | Transition timing | `150ms` |
